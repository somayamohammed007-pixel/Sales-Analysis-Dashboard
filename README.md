
# 📊 Sales & Commerce Analytics Dashboard

An interactive **End-to-End Business Intelligence Dashboard** built using 
**Power BI**,
**Power Query**, and 
**DAX** to analyze corporate sales behavior, profitability trends, and operational performance. This project delivers data-driven workforce and product analysis to optimize sales strategies.

---

## 🗺️ Dashboard Overview

### 1️⃣ Overview Page
Provides a high-level summary of the business health, including core Key Performance Indicators (KPIs), product breakdowns, and geographic segmentation.

## 🛠️ Tech Stack & Key Features

- **Data Transformation (Power Query):** Cleaned raw transactional data, optimized data types, handled null parameters, and structured the schema.
- **Data Modeling:** Established a star schema model utilizing a dedicated dynamic Calendar lookup table to power advanced time-intelligence logic.
- **DAX Calculations:** Engineered custom dynamic measures for Comparative Metrics (e.g., Year-Over-Year growth tracking for Sales, Cost, Profit, and Order volumes against Last Year 'LY').
- **Interactive UX/UI:** Embedded custom dynamic **Navigation Panels** ("Overview", "Sales & Profit Analysis", "Order Details") to ensure a seamless dashboard browsing experience.
